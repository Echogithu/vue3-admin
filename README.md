# Vue3 + Typescript + Vite

基于 Vue3-Typescript-Vite 的项目模板

## Feature

集成以下功能：

- ts
- vuex
- vue-router
- e2e
- cypress
- test unit
- jest + vtu + @testing-library/vue
- eslint + prettier
- verify git commit message
- CI
- alias
